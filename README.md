# 我的餐廳清單
屬於我的餐廳清單，可瀏覽餐廳資訊及搜尋餐廳。

## 功能
可在首頁瀏覽我的餐廳清單
點擊任一餐廳可查看詳細餐廳資訊
使用者可以餐廳名稱或餐廳類別進行搜尋

### 使用環境

需安裝 node.js 及 npm 

### 安裝

1. 開啟終端機(Terminal)，Clone 此專案至本機電腦

2. 進入專案的資料夾

3. 安裝 npm 套件

4. 安裝 nodemon 套件

5. 執行程式

6. 當terminal 出現以下字樣，表示伺服器與資料庫已啟動並成功連結
Express is listening on localhost:3000

## 開發工具

1. Node.js
2. Express 4.17.2
3. Express-Handlebars 5.3.3
4. Bootstrap 5.0
5. Font-awesome
